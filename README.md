# MAB Project

Aplikasi MAB (Monitoring Attendance and Behavior) adalah sebuah aplikasi berbasis web yang dibuat untuk memantau kehadiran dan perilaku siswa di sekolah. Aplikasi ini dibuat menggunakan teknologi Express.js, MySQL, dan EJS.

## Fitur

* Memantau kehadiran siswa
* Memantau perilaku siswa
* Mengelola data siswa
* Mengelola data guru
* Mengelola data kelas
* Mengelola data pelanggaran

## Instalasi

1. Clone repository ini
2. Install dependencies dengan menjalankan perintah `npm install`
3. Buat sebuah file `.env` dengan mengisi variabel `DB_HOST`, `DB_USER`, `DB_PASSWORD`, dan `DB_NAME`
4. Jalankan aplikasi dengan menjalankan perintah `npm run dev`
